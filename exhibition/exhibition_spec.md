\# Gh0st in the L00p — Exhibition Spec



\## Format



\- Duration: \~10 minutes

\- Output: Dual screen video

\- Screen A + Screen B

\- Same duration, same FPS



\## System



The system consists of three layers:



1\. Visual Layer

&#x20;  - StyleGAN face generation

&#x20;  - progressive degradation



2\. Text Layer

&#x20;  - machine log overlay

&#x20;  - driven by corruption\_score (0.0–1.0)

&#x20;  - state-based system



3\. Audio Layer

&#x20;  - separate voice dialogue

&#x20;  - not synchronized to text



\## Behavior



\- faces mutate continuously between A and B

\- corruption increases over time

\- text reflects system state

\- states bleed into each other



\## Goal



Create a system that feels like:

a machine continuing to operate after losing reality reference.
